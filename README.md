# 🚀 Velora AI

**Velora AI** is a modern Android application that helps users **track job applications** and **analyze resumes using AI** to improve their chances of getting hired.

---

## 📱 Overview

Velora combines:

* 📊 Job tracking system
* 🤖 AI-powered resume analysis
* 🎯 Smart career insights

All in one clean, minimal, and professional mobile app.

---

## ✨ Features

### 📌 Job Tracker

* Add and manage job applications
* Track status: Applied, Interview, Offer, Rejected
* View progress and statistics

### 🤖 AI Resume Analyzer

* Upload PDF/DOCX resumes
* Get ATS-style score (0–100)
* Detect missing keywords
* Identify weak sections
* Receive improvement suggestions

### 🧠 AI Resume Rewriter

* Improve bullet points automatically
* Generate stronger, professional wording

### 📊 Dashboard

* Total applications
* Interview rate
* Weekly progress

---

## 🛠️ Tech Stack

* **Kotlin** + Jetpack Compose
* **MVVM Architecture**
* **Hilt (Dependency Injection)**
* **Firebase (Auth, Firestore, Storage)**
* **Gemini AI API**
* **Coroutines + Flow**

---

## 🧩 Architecture

Clean architecture with separation of concerns:

* `ui/` → Screens (Compose UI)
* `viewmodel/` → Business logic
* `data/` → Firebase + repositories
* `domain/` → Models and interfaces

## 🚀 Getting Started

### 1. Get Velora AI on Goolge Play

```
https://play.google.com/store/apps/details?id=com.velora.mobile
```

### 2. Open in Android Studio

### 3. Setup Firebase

* Add your `google-services.json`
* Enable:

  * Authentication
  * Firestore
  * Storage
  * AI Logic

### 4. Add API Key

* Configure Gemini API key

---

## 🔒 Security Notes

* Do NOT upload API keys
* Use environment variables or local config
